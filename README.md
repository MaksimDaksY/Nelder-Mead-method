**Метод Нелдера - Мида**, также известный как метод деформируемого многогранника, представляет собой численный метод безусловной оптимизации функции нескольких переменных, не требующий вычисления производных. 
Основная идея метода заключается в последовательном перемещении и деформировании симплекса в пространстве переменных до приближения к точке экстремума.

Данное приложение реализует алгоритм метода Нелдера - Мида и предоставляет графический интерфейс для работы с ним.
