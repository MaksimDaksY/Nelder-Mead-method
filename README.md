**Метод Нелдера - Мида**, также известный как метод деформируемого многогранника, представляет собой численный метод безусловной оптимизации функции нескольких переменных, не требующий вычисления производных. 
Основная идея метода заключается в последовательном перемещении и деформировании симплекса в пространстве переменных до приближения к точке экстремума.

Данное приложение реализует алгоритм метода Нелдера - Мида и предоставляет графический интерфейс для работы с ним.

**Структура проекта:**
1) Program.cs - точка входа в приложение, а также основные классы для работы алгоритма Нелдера - Мида, векторов, симплекса, парсинга и вычисления целевой функции.
2) Form1.cs - основная форма приложения, предназначенная для ввода параметров метода и запуска алгоритма.
3) Form1.Designer.cs - автоматически сгенерированный код элементов интерфейса основной формы.
4) Form1.resx - ресурсный файл основной формы.
5) FormAlgorithm.cs - форма, отображающая параметры алгоритма, текущий симплекс и результаты каждой итерации метода Нелдера - Мида.
6) FormAlgorithm.Designer.cs - автоматически сгенерированный код интерфейса формы алгоритма.
7) FormAlgorithm.resx - ресурсный файл формы алгоритма.
8) Lab1.csproj - файл проекта C# с настройками сборки и зависимостями.
9) Lab1.sln - файл решения Visual Studio.
10) packages.config - список подключённых NuGet - пакетов.
11) .gitignore - список файлов и папок, исключённых из Git.
